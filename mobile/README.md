# Elektrikçiler Mobile App

Expo React Native mobil uygulaması

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
# Expo dev server'ı başlat
npm start

# Android'de çalıştır
npm run android

# iOS'ta çalıştır (sadece Mac)
npm run ios

# Web'de çalıştır
npm run web
```

## Özellikler

- ✅ Modern ve profesyonel UI/UX
- ✅ Kompakt tasarım
- ✅ Redux state management
- ✅ Expo Router navigation
- ✅ Authentication sistemi
- ✅ API entegrasyonu

## Yapı

- `app/` - Expo Router sayfaları
- `components/` - Reusable componentler
- `services/` - API servisleri
- `store/` - Redux store ve slice'lar
- `hooks/` - Custom hooks
- `utils/` - Yardımcı fonksiyonlar
- `constants/` - Sabitler (colors, API endpoints, spacing, typography)

## Notlar

- Expo SDK 52 kullanılıyor
- Tüm paketler Expo SDK ile uyumlu
- `expo-asset` gerekli paketler otomatik yüklenmiştir
